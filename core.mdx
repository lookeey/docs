## SONICSALE's core

**SONICSALE** is a *no-code, straightforward, enhanceable token launching platform*.

It is designed to get new features over time, without affecting existing launched tokens.

The **SONICSALE** offers on-chain social features and leverages the speed of the best EVM L1 chain.

## SONICSALE's features

You may [launch](/launch) a token from the [main page](https://sonic.sale), by filling up a very simple form.

You may [bump and comment on tokens](/into-the-sale).

See the links above to get more information on the details of each of these features.
